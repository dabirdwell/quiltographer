# 🎉 Rotation & Grid Snapping Added!

## New Features (Just Added):

### 1. **Rotation with R Key**
- Select any pattern
- Press 'R' to rotate 45 degrees
- Keeps rotating with each press (45°, 90°, 135°, etc.)

### 2. **Delete with Delete Key**
- Select any pattern
- Press Delete or Backspace to remove it

### 3. **Snap-to-Grid**
- Patterns now snap to a 50px grid
- Makes alignment much easier
- Professional quilt layouts!

### 4. **Better Pattern Placement**
- New patterns wrap to new rows
- No more off-screen patterns
- Organized placement

### 5. **Controls Panel**
- New panel showing keyboard shortcuts
- Visual reminder of available controls

## 🎮 Complete Controls:
- **Click pattern button** → Add to canvas
- **Click pattern** → Select it
- **Drag** → Move (snaps to grid)
- **R** → Rotate selected pattern
- **Delete** → Remove selected pattern
