## Understanding Quilting Density

### What is Density in Longarm Quilting?

**Density** refers to how closely spaced the quilting stitches are:
- **Dense (12 st/in)**: Small, tightly packed patterns - more thread, more time, stiffer quilt
- **Medium (10 st/in)**: Standard spacing - balanced drape and durability  
- **Loose (8 st/in)**: Larger patterns with more space - faster, softer drape
- **Extra Loose (6 st/in)**: Wide spacing - very fast, maximum drape

### Visual Representation
In our demo, density affects the **pattern size**:
- Dense = Smaller feathers (more repetitions)
- Loose = Larger feathers (fewer repetitions)

### Real-World Impact
| Density | Time for Queen Size | Thread Usage | Best For |
|---------|-------------------|--------------|-----------|
| Dense | 8-10 hours | 150-200 yards | Show quilts, wall hangings |
| Medium | 6-8 hours | 100-150 yards | Bed quilts, gifts |
| Loose | 4-6 hours | 70-100 yards | Utility quilts, backs |
| Extra Loose | 2-4 hours | 40-70 yards | Quick projects, practice |

### Scale vs Size
- **Scale slider**: Changes the overall pattern size (1x, 1.5x, 2x)
- **Density variant**: Changes how tightly packed the pattern is
- Combined effect: Scale × Density variant scale = final pattern size
