# PDF Samples Directory

Place your quilt pattern PDFs here for testing!

## Recommended Test Patterns:

1. **Simple Nine Patch**
   - Good for basic parsing
   - Robert Kaufman has many free ones

2. **Jelly Roll Pattern** 
   - Tests strip parsing
   - Uses pre-cuts

3. **HST Pattern**
   - Half Square Triangles
   - Complex cutting math

4. **Modern Pattern**
   - From indie designers
   - Often non-standard format

5. **Traditional Pattern**
   - From big companies
   - Standard formatting

## File Naming:
- `test-pattern.pdf` - Default test file
- `nine-patch.pdf` - Specific patterns
- `complex-cutting.pdf` - Edge cases

## Testing:
```bash
cd ..
npm run dev
```

This will parse any PDFs in this directory!
